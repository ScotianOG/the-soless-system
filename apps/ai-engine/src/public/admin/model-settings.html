Provider Information

Available Providers:

Anthropic (Claude)

The default provider using Anthropic's Claude API. Requires a valid API key.

Ollama (Phind-CodeLlama)

Local model provider using Ollama. Requires Ollama to be installed and running locally with the phind-codellama model. 

# To use Ollama provider:
1. Install Ollama from https://ollama.ai
2. Pull the model:
   ollama pull phind-codellama
3. Ensure Ollama is running locally
