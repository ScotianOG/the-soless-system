Telegram Bot Settings

How to create and use a Telegram bot:

  1. Open Telegram and search for @BotFather
  2. Send the /newbot command
  3. Follow the instructions to name your bot
  4. Copy the API token provided by BotFather and paste it above
  5. Enable the bot in the form above and click "Save Telegram Settings"
  6. Restart your server to apply the changes
  7. In Telegram, search for your bot by username and start a chat
  8. Use the /start command to begin interacting with the bot

Adding the bot to a group chat:

  1. In Telegram, create a group or open an existing group
  2. Click the group name at the top to access group settings
  3. Select "Add members" and search for your bot's username
  4. Once added, send /start in the group to activate the bot
  5. Group members can now interact with the bot by mentioning it or using commands

Note: After making changes to Telegram settings, you need to restart the server for changes to take effect.
