Website Integration

Integrate the SOLess chatbot into your website by adding the widget script to your HTML.

Simply copy the code below and paste it before the closing </body> tag in your HTML.

Widget Code

This will add a chat button to the bottom right of your website.

Configuration Options

You can customize the widget by modifying the SOLessConfig object:

  • position: 'bottom-right' (default), 'bottom-left', 'top-right', 'top-left'
  • primaryColor: Change to match your website's branding (hex color code)
  • welcomeMessage: The initial message shown when the chat is opened

Example custom configuration:

window.SOLessConfig = {
  position: 'bottom-left',
  primaryColor: '#2196F3',
  welcomeMessage: 'Welcome to SOLess! How can I help you?'
};
